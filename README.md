# *Visualg*

💻 Exercícios de Lógica de Programação (Visualg)
Este repositório contém uma coleção de algoritmos desenvolvidos em Portugol utilizando a ferramenta Visualg. Os exercícios foram realizados durante o curso de Algoritmo do portal Curso em Vídeo, ministrado pelo professor Gustavo Guanabara.

#
🚀 Objetivo
O foco principal destes exercícios foi consolidar a base necessária para qualquer linguagem de programação, praticando:

* Estruturas de controle (Condicionais e Repetição).

* Manipulação de Variáveis e Operadores Aritméticos/Lógicos.

* Procedimentos, Funções e Passagem de Parâmetros.

* Estruturas de dados (Vetores e Matrizes).

#
📂 Conteúdo do Repositório

Os arquivos .alg cobrem diversos desafios, divididos por complexidade:

🔹 Fundamentos e Condicionais

CARTEIRADEMOTORISTA.ALG: Verificação de idade e permissão.

CONVERSORDEMOEDA.ALG: Cálculos matemáticos básicos.

APROVEITAMENTOALUNO.ALG: Cálculo de médias e notas.

🔹 Estruturas de Repetição (Loops)

CONTADOR.ALG e SUPERCONTADOR.ALG: Prática com Enquanto, Repita e Para.

FATORIAL.ALG: Cálculo matemático utilizando laços.

SELETORDEPESSOAS.ALG: Filtragem de dados com estruturas de repetição.

🔹 Vetores e Matrizes (Estruturas de Dados)

TESTEVETOR.ALG e TURMAVETOR.ALG: Manipulação de listas de dados.

ORDENAVETOR.ALG: Implementação de lógica de ordenação.

MATRIZIDENTIDADE.ALG e VALORESMATRIZ.ALG: Operações avançadas com tabelas multidimensionais.

🔹 Funções e Procedimentos

SOMAFUNCAO.ALG: Uso de funções com retorno.

SOMAPARAMETROSREFERENCIA.ALG: Entendimento de escopo e passagem por referência.


#
🛠️ Como rodar os exercícios
Baixe e instale o Visualg (versão 3.0 recomendada).

Clone este repositório:

Bash

git clone https://github.com/Pedro-Ramon2608/Visualg.git

Abra o Visualg, vá em Arquivo > Abrir e selecione qualquer arquivo .alg.

Aperte F9 para executar o algoritmo.
